# JARVIS_Voice_Assistant_system

JARVIS is a Python-based smart voice assistant designed to interact with users through natural speech. It uses speech recognition to understand commands and text-to-speech (TTS) to respond, creating a hands-free experience similar to Iron Man‚Äôs JARVIS.

The goal of this project is to build an intelligent, voice-controlled assistant that offers a seamless and interactive user experience inspired by the fictional JARVIS system.


## üõ† Features

This assistant is capable of performing various tasks, including:

- Opening applications based on voice commands

- Searching for information on Google or Wikipedia

- Playing random music

- Telling jokes and engaging in small talk

- Providing time, date, or weather information

- Executing system automation commands

- Greet the user according to the time of day (morning, afternoon, evening)

- Recognize voice commands using Google Speech Recognition

- Speak responses using pyttsx3

- Time & Date announcements

- Wikipedia search with spoken summary

- Open websites like Google, Facebook, YouTube

- Play random music from a specified folder

- Open system applications: Calculator, Notepad, CMD

- Open Calendar (Google Calendar via browser)

- Tell jokes and respond to basic small talk

- Exit gracefully with a voice command

- Python 3.11 or higher 


## How to run? 

1. Create a virtual environment:

   ```bash
   conda create --prefix .\jarvis python=3.11 -y
   ```

2. Activate virtual environment:

   ```bash
   conda activate .\jarvis
   ```

3. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the JARVIS script:

   ```bash
   python jarvis.py
   ```
